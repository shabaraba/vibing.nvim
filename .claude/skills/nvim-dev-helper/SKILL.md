# Neovim 開発ヘルパースキル

このスキルは Neovim プラグイン開発に特化したサポートを提供します。

## 機能

### 1. Lua API の最適な使用
- `vim.api.*` vs `vim.*` の適切な選択
- 非同期処理（`vim.schedule`, `vim.loop`）の実装
- バッファ/ウィンドウ/タブページ操作のベストプラクティス

### 2. プラグイン構造の最適化
- モジュール構成の提案
- 遅延読み込みの実装
- パフォーマンス最適化

### 3. テストとデバッグ
- `plenary.nvim` を使ったテスト作成
- `:lua vim.inspect()` を使ったデバッグ
- エラーハンドリングのパターン

### 4. ドキュメント生成
- `:help` ドキュメント（vimdoc）の生成
- README の自動更新
- 型アノテーション（LSP 対応）

### 5. MCP サーバー統合
- vibing-nvim MCP ツールの効果的な使用
- RPC サーバーとの連携パターン
- 非同期 I/O のベストプラクティス

## 使用例

```vim
" チャット内で自動的に利用可能

```
